# output-stream

## 职责

从环形缓冲拉取采集数据并写入输出文件，支持 cross 模式解码。

## 接口定义（可选）

| 函数/方法 | 参数 | 返回值 | 说明 |
|----------|------|--------|------|
| runRawWriter | ctx | void | 启动写线程流程 |

## 行为规范

### 数据写出
**条件**: 采集进行中且 ring buffer 有数据
**行为**: 按目标大小/连续模式写入文件
**结果**: 写入完成或标记失败并请求停止

## 依赖关系

```yaml
依赖: output-ringbuffer
被依赖: capture
```
