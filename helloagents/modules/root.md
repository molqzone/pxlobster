# root

## 职责

提供 libusb 的 C 绑定入口，供 USB 相关模块共享使用。

## 接口定义（可选）

| 函数/方法 | 参数 | 返回值 | 说明 |
|----------|------|--------|------|
| libusb | - | C 绑定 | 导出 libusb 头文件绑定 |

## 行为规范

### 绑定暴露
**条件**: 需要访问 libusb API
**行为**: 通过 `@cImport` 导出头文件符号
**结果**: 供 `usb` 模块调用

## 依赖关系

```yaml
依赖: libusb.h
被依赖: usb
```
