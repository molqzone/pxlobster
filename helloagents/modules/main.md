# main

## 职责

CLI 入口：解析参数、分发 scan/prime-fw/capture 命令，并统一处理输出与退出码。

## 接口定义（可选）

| 函数/方法 | 参数 | 返回值 | 说明 |
|----------|------|--------|------|
| main | - | !void | 解析参数并执行命令 |

## 行为规范

### 命令分发
**条件**: 解析得到 scan/prime_fw/capture 子命令
**行为**: 调用对应流程，并在 `--verbose` 时输出调试日志
**结果**: 成功输出结果，参数错误返回退出码 2

## 依赖关系

```yaml
依赖: args, capture, usb, device
被依赖: 无（入口模块）
```
