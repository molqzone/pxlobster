## 3. 项目概述

### 核心功能
- 扫描 PXLogic 设备（`--scan`）
- 注入固件/位流（`--prime-fw`）
- 停止正在进行的采集（`--stop`）
- 采集输出 `bin` / `sr`，支持采样率与触发配置
- 支持 stdout 管道输出（`--stdout`）

### 项目边界
```yaml
范围内:
  - PXLogic 设备扫描、固件注入与采集控制
  - 采集数据输出到 bin/sr 或 stdout
范围外:
  - GUI 前端与可视化分析
  - 设备固件开发与硬件设计
```
